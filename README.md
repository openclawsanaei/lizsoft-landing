# Lizsoft Landing Page

Simple, professional one-page site for Lizsoft. Built with plain HTML5, CSS3, and vanilla JavaScript.

## Structure

- `index.html` — Semantic HTML
- `styles.css` — Mobile-first responsive styles
- `script.js` — Minimal smooth scrolling and interactions

## Development

Open `index.html` in a browser to preview.

## Deployment

This repository is configured for GitHub Pages deployment from the `/docs` folder.
